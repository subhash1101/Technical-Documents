# Web & SQL Basics – Q&A

## 1. What is SQL?

SQL (Structured Query Language) is a language that is used to operate on databases.
It is used to store, retrieve and update data in a database.

---

## 2. Void Elements in HTML

Void elements are HTML elements that do not have a closing tag.

Examples: `<img>, <br>, <hr>, <input>, <meta>`.

---

## 3. Suru Abhishek – Difference Between SQL and MySQL

SQL
* SQL is a language used to manage database.
* Used to write queries.

MySQL
* MySQL is a database management system that uses SQL.
* Used to store and manage data.

---

## 4.Values of justify-content

justify-content aligns flex items according to the main axis.

Common values:

* flex-start
* flex-end
* center
* space-between
* space-around
* space-evenly

---

## 5. Difference Between Class and ID

Class
* Used for multiple elements.
* Starts with "." in CSS.

ID
* Used for one unique element.
* Starts with "#" in CSS.

---

## 6."alt" Property in <img> Tag

The alt attribute provides alternative text for an image if the image cannot be displayed.

Example:

```html
<img src="cat.jpg" alt="A cute cat">
```

---

## 7. Inline vs Block Elements

Inline Element
* Does not start on a new line.
* Example: `<span>, <a>`

Block Element
* Starts on a new line.
* Example: `<div>, <p>`

---

## 8.Triggers in SQL

A **trigger** is a special procedure that automatically runs when an event occurs on a table (INSERT, UPDATE, DELETE).

---

## 9. Difference Between Relative and Absolute Position

Relative
* Positioned relative to its normal position.
* Element still occupies space.

Absolute
* Positioned relative to the nearest positioned ancestor.
* Not in the page it is outside the alignment of page
  
---

## 10. UNION vs UNION ALL

UNION
* Removes duplicate rows.
* Slower due to duplicate checking.

UNION ALL
* Keeps duplicate rows.
* Faster than "UNION ALL"
---

## 11.Import Data from CSV to SQL Table

```sql
COPY table_name FROM 'file.csv' DELIMITER ',' CSV HEADER;
```

---

## 12.Properties of Meta Tags

Meta tags provide metadata about the HTML document.

Example:

```html
<meta charset="UTF-8">
<meta name="description" content="Web tutorial">
```

---

## 13.Properties of Flexbox

Common Flexbox properties:
* display: flex
* flex-direction
* justify-conent
* align-items
* flex-wrap
---

## 14. Difference Between space-around and space-between

space-around
* Equal space around each item. 
* Space exists at the edges.

space-between
* Space between items.
* No space at the edges.
